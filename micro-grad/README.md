# micrograd

自动微分引擎教学实现，包含核心组件演示和完整训练示例（**不支持张量**）。

## 文件说明
| [`micrograd.ipynb`](./micrograd.ipynb) | 逐步拆解：从 Value 基础 → Neuron → Layer → MLP，每步都有计算图可视化 |

| [`micrograd.py`](./micrograd.py) | 完整演示：用 MLP 训练 100 步梯度下降，展示真实训练流程 |

## 功能介绍

### Value（标量自动微分）
- 支持基本运算：加、减、乘、除、幂
- 实现反向传播和梯度计算
- 构建计算图并可视化

### Neuron（单个神经元）
- 线性变换 + tanh 激活
- 参数：权重 w，偏置 b

### Layer（神经网络层）
- 多个神经元并行
- 灵活的输入输出维度

### MLP（多层感知机）
- 多层串联网络
- 支持任意层数和神经元数
